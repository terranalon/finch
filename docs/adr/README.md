## Creating a New ADR

1. Copy the format from an existing ADR
2. Use the next sequential number: `NNNN-title-with-dashes.md`
3. Fill in all sections, especially Context and Consequences
4. Submit PR for review
5. Update this index after approval

## ADR Status

- **Proposed**: Under discussion, not yet approved
- **Accepted**: Decision made, ready for implementation
- **Deprecated**: No longer relevant
- **Superseded**: Replaced by another ADR
- **Rejected**: Considered but not adopted
